## Customize configuration

See [Vite Configuration Reference](https://vitejs.dev/config/).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Deploy Flow

```sh
npm run build
git add dist -f
git commit -m <your commit message>
git subtree push --prefix dist origin gh-pages
```
